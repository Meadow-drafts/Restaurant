## Usage

This project was built with [Vue 3](https://v3.vuejs.org/) and [Vite](https://vitejs.dev/).
It is meant to manage a restaurant system

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run dev
```

#### Compiles and minifies for production
```
npm run build
```
